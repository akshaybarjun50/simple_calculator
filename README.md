# 🧮 Simple Calculator (Python CLI)

A simple command-line calculator built with **Python** that performs basic mathematical operations. This project is beginner-friendly and demonstrates the use of loops, conditional statements, exception handling, user input, and the `math` module.

---

## 📌 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 🧮 Modulus
* 🔢 Power (Exponentiation)
* √ Square Root
* 🚪 Exit option
* ✅ Input validation using `try-except`
* ⚠️ Handles division/modulus by zero
* 🎨 Colored terminal output (ANSI escape codes)

---

## 🛠️ Technologies Used

* Python 3
* `math` module (built-in)

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/akshaybarjun50/simple_calculator.git
```

2. Navigate to the project folder:

```bash
cd simple_calculator
```

3. Run the program:

```bash
python calculator.py
```

> Make sure Python 3 is installed on your system.

## ⚠️ Error Handling

The program handles the following errors gracefully:

* Invalid menu selections
* Non-numeric input
* Division by zero
* Modulus by zero
* Square root of a negative number
* Keyboard interruption (`Ctrl + C`)
